REST API calls (The Heart of the Application)

REST API

This REST API provides all the essential API calls for the whole application. Just by Looking at each of the API calls, one can understand the the whole functionality of the appication.

The REST API calls return a JSON object with detailed HTTP Response Codes (200, 404 etc..) along with DATA Array and a message (in case of error) 

{
     "code":   200,
     "data":   ["SFO-AMSTERDAM.json", "SFO-DELHI.json", "SFO-PARIS.json"],
     "message": ""
}

  • Carriers : All the CARRIERS availble
  • URL for ALL Carriers    http://localhost:3000/carriers
  • Flights : All the FLIGHTS for a given Carrier
  • URL    http://localhost:3000/SouthWest
  • URL    http://localhost:3000/DeltaAirlines
  • for Error Condition    http://localhost:3000/DeltaAirlines1234
  • Specific FLIGHT information for a given Carrier
  • URL    http://localhost:3000/SouthWest/SFO-DELHI
  • URL    http://localhost:3000/DeltaAirlines/OAK-AMS.json
  • for Error Condition    http://localhost:3000/DeltaAirlines/abc1234
